## 命令一覧
|command|brainf*ck|Description|
|:-----:|:-------:|:---------:|
|ﾝｺﾞ|>|ポインタを1つ進める|
|ﾝｺﾞwww|<|ポインタを1つ戻す|  
|ﾝｺﾞｺﾞ|+|ポインタが指す値を1増やす|
|ﾝｺﾞｺﾞwww|-|ポインタが指す値を1減らす|
